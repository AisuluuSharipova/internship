Course Descriptions

Introduction to Engineering and Computer Science

First part of the course introduces students to the profession, including the disciplines of chemical, civil, computer, electrical, environmental, and mechanical engineering. Prepares students for success through the integration of technical problem solving and engineering design, ethical decision-making, teamwork, and communicating to diverse audiences. Second part is aimed at students with little or no programming experience using Python programming language.

Teachers: Mohammad Tauheed Khan, Burul Shambetova

CS 100.1: Introduction to Programming (Python)

Introduces students programming language Python by means of well chosen problems, exercises and practice laboratories.

Teachers: Burul Shambetova, Bekmamat Tashmatov

COM 103: Programming Languages I (Python)

This subject is aimed at students with little or no programming experience. It aims to provide students with an understanding of the role computation can play in solving problems and help students feel confident in their ability to write small programs that accomplish useful goals.

COM 103: Programming Languages I (Java)

Introduction to Java programming language covering basic syntax, data types, control structures, and object-oriented programming fundamentals.

COM 103: Programming Languages I (C++)

Introduction to C++ programming fundamentals, covering variables, control structures, functions, arrays and strings. Develops problem-solving skills and provides foundation for further programming studies.

COM 108: Engineering Computer Graphics

Examines internet technology and website design/development process. Covers HTML5, CSS3, and building dynamic interactive web applications using databases.

Teachers: Andrei Ermakov, Ruslan Isaev

COM 990: Computer Literacy

Designed for individuals seeking to develop fundamental computer literacy skills. Covers computer operation basics, internet usage, productivity tools, email communication, document creation, and data management.

Teacher: Zhenishbek Orozakhunov

CS 103: Calculus - 1

First course in calculus of one variable for computer science and engineering students. Topics include review of polynomials, trigonometric, exponential, and logarithmic functions, limits, derivatives, and applications of differential calculus to real-world problems.

Teacher: Khaled Mohammad

CS 104: Calculus - 2

Second course in calculus covering integration, basic techniques for integration, applications of integration, and introduction to differential equations.

Teachers: Tahir Aslanol, Khaled Mohammad, Esen Rakhmanov

CS 105: Physics I - Mechanics

Course covering Newton's Mechanics principles and applications in engineering contexts.

CS 106: Physics II - Electricity and Magnetism

Study of electric and magnetic phenomena, their laws and applications in devices like radios, televisions, electric motors, and computers. Covers inter-atomic and intermolecular forces electric in origin.

Teachers: Ruslan Isaev, Khaled Mohammad

CS 1st Year English Language 1 & 2

Comprehensive course designed for learners at all levels to master English. Develops strong foundational skills in reading, writing, listening, and speaking. Enhances vocabulary, grammar proficiency, pronunciation, and conversational abilities while cultivating critical thinking through literature and media.

Turkish Language COM

Introduction to Turkish language covering basic grammar, vocabulary, reading, writing, and conversational skills for computer science students.

Teacher: Zulfizar Murzaeva

COM 141: Programming Languages II (Java)

This course offers a comprehensive introduction to Java, focusing on essential programming concepts and hands-on experience. Students explore Java Basics, Methods, Strings, ArrayLists, Sets, HashMaps, and File Handling with a culminating project.

Teacher: Zhenishbek Orozakhunov

COM 141: Programming Languages II (Python)

Advanced course on Python covering OOP, packaging, virtual environments and ML/AI projects development.

Teachers: Alymbek Ergeshov, Ruslan Isaev, Mekia Shigute Gaso, Dim Shayahmetov

COM 206: Electrotechnics and Electronics

Teaches direct current circuit analysis techniques and introduces basic circuit design with Arduino-based applications.

Teachers: Mohammad Tauheed Khan, Eyup Semsi Yilmaz

COM 225: Competitive Programming

Introduces fundamentals of competitive programming, focusing on algorithmic thinking and coding proficiency. Covers data structures, algorithms, dynamic programming, graph theory, and mathematical concepts through coding exercises and contests.

Teacher: Zhenishbek Orozakhunov

COM 230: Front-end Development

Comprehensive web development course covering HTML, CSS, JavaScript, and modern front-end frameworks for creating responsive web applications.

COM 235: Object Oriented Programming (Java)

Provides introduction to principles and practices of object-oriented programming. Students learn to design, implement, and test object-oriented programs using Java, covering classes, objects, inheritance, polymorphism, encapsulation and software design principles.

CS 202: Data Structures

Fundamental course covering various data structures including arrays, linked lists, stacks, queues, trees, graphs, and hash tables, along with their applications and algorithm analysis.

CS 203: Web Technologies

Examines internet technology and website design/development process. Covers HTML5, CSS3, and building dynamic interactive web applications using databases.

Teacher: Ruslan Isaev

CS 204: Back-end Development with Spring Framework

Introduces Java Spring framework for enterprise solutions. Covers dependency injection, Gradle, databases with Spring and Hibernate, web applications with Spring MVC, Spring Security, JDBC, web forms, and aspect-oriented programming.

Teacher: Zhavlon Khamidov

CS 205: Database Systems

Introductory database course covering database systems role in information systems, database development in SDLC, data modeling concepts (E-R and Class diagrams), SQL queries, DBMS (MS SQL Server), and data integrity issues.

Teachers: Nargiza Zhumalieva, Zhavlon Khamidov

CS 206: Digital Electronics

Fundamental course introducing digital circuits, logic gates, flip-flops, counters, and registers. Helps understand how digital signals are processed and controlled.

Teacher: Mohammad Tauheed Khan

ELT.123: Electronic Components and Circuits

Introduction to DC theory, analysis and design of electric circuits covering voltage, current, power, energy, resistance, capacitance, inductance, Kirchhoff's laws, node analysis. Includes Arduino-based circuit design, programming and applications.

Teacher: Mohammad Tauheed Khan

Discrete Mathematics

Fundamental mathematical course covering discrete structures including sets, relations, functions, logic, combinatorics, graph theory, and algebraic structures essential for computer science applications.

Introduction to Probability and Statistics

Course covering fundamental concepts of probability theory, statistical analysis, data interpretation, and their applications in computer science and engineering fields.

COM 302: Artificial Intelligence

Covers fundamentals of AI including intelligent agents, problem-solving, knowledge representation, reasoning, and machine learning concepts. Students develop AI applications in Machine Learning, Neural Networks, Deep Learning, and Reinforcement Learning using Python or MATLAB.

Teachers: Arslan Khan, Selcuk Cankurt

COM 304: Operating Systems

Introductory course covering basic concepts of operating systems, resource management (memory, peripherals, CPU scheduling), system call interface, process creation and synchronization, application communication, memory hierarchy, virtual memory, and file management.

COM 308: Computer Architecture and Operating Systems

Comprehensive course covering both computer architecture principles and operating system concepts including processor design, memory hierarchy, and system software interaction.

Teacher: Talgat Mendekov

COM 416: Machine Learning

Covers main models and algorithms for regression, classification, clustering, and probabilistic classification. Focuses on data handling, visualization, and model development in domain-based problem-solving using Python.

COM 427: Front-end Development

Covers web development basics with HTML, CSS, JavaScript programming, jQuery JavaScript library, and Bootstrap framework. Includes projects ranging from simple HTML pages to complete JavaScript-based Chrome extensions.

Teacher: Ruslan Isaev

COM/MAT 622: Object Oriented Programming (Python)

Introduces principles and practices of OOP using Python through lectures, hands-on coding, and capstone project. Covers classes, objects, inheritance, polymorphism, encapsulation, GUI development with PyQt, unit tests, and SOLID principles.

Teachers: Erustan Erkebulanov, Zhenishbek Orozakhunov

CS 301: Design and Analysis of Algorithms

Advanced course covering algorithm design techniques, complexity analysis, and efficiency optimization for various computational problems.

CS 303: Computer Networks & Telecommunications

Covers telecommunications networks, data exchange between computers, network links (cable and wireless media), network nodes, and applications like WWW, shared resources, email, and instant messaging.

Teachers: Mohammad Imtiyaz Gulbarga, Aziret Ramankulov

CS 303: Mobile App Development

Examines principles of mobile application design and development on Android platform. Covers memory management, UI design, input methods, data handling, network techniques, GPS, motion sensing, and project deployment.

Teachers: Nurmuhammet Abdullayev, Ruslan Isaev

CS 308: Computer Organization & Architecture

Provides broader understanding of computer systems, organization and architecture. Covers essential computer working principles, logical and arithmetic operations, and fundamental building blocks.

Software Engineering I

Fundamental course covering software development methodologies, requirements analysis, system design, implementation, testing, and maintenance processes in software engineering.

Teachers: Mekia Shigute Gaso, Remudin Reshid Mekuria, Nargiza Zhumalieva

Cybersecurity

Comprehensive course covering principles of information security, network security, cryptography, threat analysis, and defense mechanisms to protect computer systems and networks from cyber threats.

Ethical Hacking & Penetration Testing

Hands-on course teaching ethical hacking techniques, vulnerability assessment, penetration testing methodologies, and security auditing to identify and address system vulnerabilities.

Teachers: Mohammad Imtiyaz Gulbarga, Akzhol Suborov, Sabina Sulaimanova

Psychology (COM-20)

Introduction to psychological principles and their application in human-computer interaction, user experience design, and understanding user behavior in technological contexts.

Teacher: Fatma Hena

COM 231: Design Thinking for Product Solutions

Course focusing on human-centered design approach for developing innovative technology products and solutions through empathy, ideation, prototyping, and testing.

Teacher: Andrei Ermakov

COM 232: Robotics

Introduction to robotics principles including robot design, control systems, sensors, actuators, and programming of robotic systems for various applications.

Teacher: Artur Kutlugildin

COM 233: Data Analysis and Visualization

Course covering data analysis techniques, statistical methods, and visualization tools for interpreting and presenting complex data in meaningful ways.

Teachers: Mekia Shigute Gaso, Remudin Reshid Mekuria

COM 249: Interpersonal Communication in IT

Helps IT students build interpersonal communication skills for diverse teams, conflict management, and technical concept communication. Develops assertiveness, influence, and conflict resolution skills for IT environments.

COM 312: Art and Design

Prepares students to think critically and execute visual solutions across print, digital platforms and motion graphics. Explores design process from industry professionals and graduates students with well-rounded portfolios.

Teacher: Andrei Ermakov

COM 405: Introduction to Data Mining

Fundamental course covering data mining techniques, pattern discovery, association rule mining, classification, and clustering algorithms for extracting knowledge from large datasets.

COM 408: Metrology Standard & Certification

Teaches analysis of measurement systems for process stability and capability. Covers continuous measurement systems characterization, statistical control, capability assessment, and discrete measurement analysis using Python.

Teachers: Ruslan Isaev, Artur Kutlugildin

COM 410: Philosophy of Technologies

Examination of philosophical questions raised by technology, including ethical implications of technological development, impact on society, and the relationship between humans and technology.

Teacher: Arslan Khan

COM 425: State Examinations

Comprehensive state examination designed to assess learning outcomes in key Computer Science disciplines including Programming languages, Web programming (Backend development with Spring Boot) and Software Engineering.

Teachers: Ermek Doszhanov, Ruslan Isaev, Zhavlon Khamidov, Nurlan Shaidullaev, Burul Shambetova, Dim Shayahmetov

COM 431: Image Processing

Course covering digital image processing techniques including image enhancement, restoration, compression, segmentation, and pattern recognition algorithms.

Teachers: Mohammad Tauheed Khan, Eyup Semsi Yilmaz

COM 476: Data Mining and Storage

Advanced course covering data mining concepts and techniques along with modern data storage solutions, big data technologies, and distributed computing frameworks.

CS 403.1: Software Architecture & Design Patterns

Introduction to architecture and design of complete software systems using components and patterns. Covers architectural principles, design documentation, and relationships between abstraction levels.

CS 406: Thesis Diploma

Final year thesis project under faculty supervision, requiring students to demonstrate comprehensive knowledge and research skills in computer science.

Supervisors: Andrei Ermakov, Gulzada Esenalieva, Mohammad Imtiyaz Gulbarga, Ruslan Isaev, Arslan Khan, Mohammad Tauheed Khan, Azamat Kibekbaev, Burul Shambetova

CS 409: Human Computer Interaction

The study of human-computer interaction enables system architects to design useful, efficient, and enjoyable computer interfaces. This course teaches the theory, design procedure, and programming practices behind effective human interaction with computers, with particular focus on smart phones and tablets. Covers interaction design, implementation, evaluation, usability engineering, prototyping, and novel interfaces beyond traditional graphical user interfaces.

Teacher: Burul Shambetova

Math-23: Robotics

Mathematical foundations for robotics including kinematics, dynamics, control theory, and motion planning algorithms.

Teacher: Curt Schurgers

MDE 402: Philosophy of Tech

Examination of philosophical questions raised by technology, including ethical implications of technological development, impact on society, and the relationship between humans and technology.

Teacher: Arslan Khan

MDE 404: Manas Taanuu

Online course focusing on the study of Manas epic, Kyrgyz cultural heritage, and its significance in contemporary society.

Fundamentals of Scientific Research

Course designed to foster and evaluate students' skills necessary for conducting and reporting research. Covers research idea development, literature review, research questions formulation, data collection, ethical guidelines, and research proposal writing with emphasis on meeting deadlines and oral communication of findings.

Teacher: Burul Shambetova

VR Design

Comprehensive course in virtual reality design covering 3D environment creation, user interaction design, immersive experience development, and VR application implementation using modern tools and technologies.

Teachers: Ruslan Isaev, Ruslan Migranov

Internships

Three main internship sections: Educational internship, Industrial internship, and Pre-qualificational internship. Requirements vary by study year, providing practical experience in real-world computing environments.

COM 000: BootCamp

Intensive training program covering various computer science topics through hands-on projects and accelerated learning sessions.

ENGLISH LANGUAGE Present Academic year

Current academic year English language course focusing on continuous improvement of language skills relevant to computer science professionals, including technical writing and presentation skills.

Teacher: Murray James Eldred

Faculty Assessment Simple Test

Assessment test designed to evaluate student progress and faculty teaching effectiveness across computer science program, ensuring quality education standards.

Teacher: Ruslan Isaev

Training program for Moodle managers

Specialized training program for managing Moodle learning management system, covering course creation, user management, assessment tools, and platform administration for effective online education delivery.

Teachers: Ruslan Isaev, Nurlan Shaidullaev

Turkish Language COM (A. Beisheeva)

Advanced Turkish language course focusing on technical vocabulary and communication skills specific to computer science context, including professional terminology and business communication.

Teacher: Aigul Beisheeva
